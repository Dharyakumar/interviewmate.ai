# InterviewMate AI 🚀

InterviewMate AI is an interview preparation platform that helps students and job seekers practice mock interviews and receive structured feedback.

## Features

- Landing Page
- Mock Interview Flow
- Multiple Interview Questions
- Answer Collection
- Score Calculation
- Interview Summary Page

## Tech Stack

- HTML
- CSS
- JavaScript
